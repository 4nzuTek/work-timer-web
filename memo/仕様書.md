# 作業時間計測タイマー 仕様書

## 1. プロジェクト概要

ブラウザ上で動作する作業時間を計測し、カレンダー形式で履歴を管理するタイマーアプリケーション。

## 2. 目的

作業時間を計測し、日付ごとに「何時から何時まで何の作業をしたか」を記録・確認できるツール。特定の日に何時間何の作業をしたかを把握することを主目的とする。

## 3. 機能要件

### 3.1 タイマー機能

#### 3.1.1 タイマーの操作
- **開始**: タイマーを開始する（開始時刻を自動記録）
- **停止**: タイマーを停止し、計測時間を記録する（終了時刻を自動記録）
- 同時に複数のタイマーを実行することは想定していない（1つのタイマーのみ）

#### 3.1.2 作業内容の設定
- タイマー開始時に、作業内容（名前/説明）を入力できる
- 作業内容の入力は任意（必須ではない）

#### 3.1.3 タグ機能
- タイマー開始時に、タグを付けることができる
- 複数のタグを同時に設定可能
- タグは事前登録されたものから選択する形式

### 3.2 履歴管理機能

#### 3.2.1 記録内容
- 日付
- 開始時刻（時:分）
- 終了時刻（時:分）
- 作業時間（時間:分）
- 作業内容（名前/説明）
- タグ（複数）

#### 3.2.2 カレンダー表示
- **月間カレンダー形式**で作業履歴を表示
- 特定の日をクリックすると、その日の作業記録の詳細を確認できる
- 同じ日に複数の作業記録が存在する場合、すべて表示する
- 例: 10/28は「10:00-12:00 ○○の作業」「14:00-16:30 ××の作業」のように複数表示

### 3.3 表示機能

#### 3.3.1 日別表示
- 特定の日に何時間、何の作業をしたかを一覧表示
- 作業内容とタグを表示

### 3.4 記録管理機能

#### 3.4.1 編集・削除機能
- 記録後の作業記録を編集できる
- 記録後の作業記録を削除できる
- 編集可能項目: 作業内容、開始時刻、終了時刻、タグ

### 3.5 タグ管理機能

#### 3.5.1 タグの登録・管理
- タグを事前に登録できる
- タグの削除機能（既に使用されているタグの扱いは未確定）

## 4. データ保存

### 4.1 保存方法
- **ブラウザのローカルストレージを使用**
- 作業記録、タグ一覧をローカルストレージに保存
- ブラウザを閉じてもデータは保持される

## 5. UI要件

### 5.1 レイアウト
- **タイマー操作エリア**: タイマーの開始・停止、作業内容入力、タグ選択
- **カレンダー表示エリア**: 月間カレンダーを表示
- **作業記録一覧表示エリア**: 選択した日の作業記録一覧を表示（編集・削除機能含む）
- **タグ管理エリア**: タグの登録・削除（必要に応じて）

### 5.2 デザイン
- シンプルで使いやすいUI
- モダンなデザイン

## 6. 技術要件

### 6.1 実装技術
- HTML5
- CSS3
- JavaScript（バニラJSまたは軽量ライブラリ）

### 6.2 動作環境
- モダンブラウザ（Chrome, Firefox, Safari, Edge）
- JavaScript有効
